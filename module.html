Overview

Data fitting is a really, really big deal in many disciplines. We will start this module by looking at the gold-medal winning long jump performances in the Olympics since 1900. And we’ll finish this module by asking you to interact more with the application, Keeling Curve, Revisited. Here are the questions that this topic generates

When we have lots and lots of data, how do we approximate the data with a linear function? How do we measure the “total error”? How do we know how good a fit to the data our linear function is?

We’ve got an extra “Resources” tab for this module. If you are curious, check out the Kinematics of the Long Jump article, and a video talking about Bob Beamon’s amazing jump in the 1968 Olympics. There’s also a spreadsheet with the data that we used in the video, plus some instructions on how to create a line of best fit through other that data plus other data sets.
